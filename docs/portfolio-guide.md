# Portfolio Guide

## 1. Mục đích

Tài liệu này giúp bạn biến dự án thành một portfolio entry có thể đưa lên GitHub, CV và LinkedIn.

## 2. Nội dung cần nổi bật

- Problem statement rõ ràng
- AI pipeline end-to-end
- Clean Architecture backend
- React dashboard đẹp và responsive
- Docker deployment
- Test strategy
- Documentation đầy đủ

## 3. Cách trình bày trên GitHub

- README ngắn gọn, có quick start và architecture overview
- Có ảnh demo hoặc GIF nếu sẵn
- Có roadmap và phase docs
- Có link đến deploy guide và API docs
- Có mục limitations và future work

## 4. Cách trình bày trên CV

- 1-2 bullet về mục tiêu dự án
- 1 bullet về AI pipeline
- 1 bullet về backend/frontend/deploy
- 1 bullet về testing hoặc architecture

### Ví dụ

- Built a Smart Parking Management System using YOLO-based license plate detection, PaddleOCR, FastAPI, React and MySQL.
- Designed a Clean Architecture backend with JWT auth, audit logging, and parking session management.
- Containerized the system with Docker Compose and documented deployment, testing, and demo workflows.

## 5. Cách trình bày khi phỏng vấn

- Bắt đầu bằng bài toán và lý do chọn nó
- Giải thích architecture trước code details
- Nói rõ trade-off: vì sao OCR dùng PaddleOCR
- Nêu thử thách: dữ liệu, detection, integration, deployment
- Kết thúc bằng future work

## 6. Checklist trước khi public

- [ ] README có quick start
- [ ] Không commit secret
- [ ] Link docs không bị gãy
- [ ] Build và test pass
- [ ] Có demo script
- [ ] Có interview notes
- [ ] Có deploy guide
