# Frontend

React + TypeScript frontend for Smart Parking Management System.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Set `VITE_API_BASE_URL` if the backend is not running on `http://localhost:8000/api/v1`.

## Pages

- Login
- Dashboard
- Camera Live
- Vehicle
- Parking History
- Statistics
- Settings
- User Management
